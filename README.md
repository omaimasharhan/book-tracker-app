# Book Tracker App

A simple React Book Tracker app that helps users keep track of their reading progress.

## Features

- Add a book title
- Enter the number of pages
- Enter the number of chapters
- Track:
  - Pages read
  - Chapters read
  - Reading sessions
- Increment and decrement progress using buttons
- Reset progress
- Prevent pages and chapters from going below 0 or above the total amount entered
- Conditional rendering message when no progress has been added yet

## Technologies Used

- React
- JavaScript
- CSS
- Vite

## Components

- `BookForm`
- `Button`
- `StatisticLine`

## React Concepts Used

- `useState`
- Props
- Event handlers
- Conditional rendering
- Multiple components
